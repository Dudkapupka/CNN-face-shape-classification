# Классификация формы лица с использованием сверточных нейронных сетей

Работа посвящена задаче классификации формы лица по изображениям с применением методов глубокого обучения. Рассматривается построение и анализ собственных архитектур сверточных нейронных сетей (CNN), ориентированных на различение визуально близких классов без использования предобученных моделей. Основной целью является исследование влияния архитектурных и обучающих решений на качество классификации.

---

## Задачи работы

- формализовать задачу классификации формы лица по изображению;
- реализовать базовую сверточную нейронную сеть и зафиксировать стартовые метрики;
- последовательно усложнить архитектуру модели и проанализировать влияние изменений;
- исследовать применение механизмов внимания и современных функций активации;
- оценить качество моделей с использованием стандартных метрик классификации;
- провести анализ ошибок и устойчивости полученных решений.

---

## Используемый датасет

В работе используется датасет изображений лиц, представленных в формате RGB, с разметкой по классам формы лица:  
**Heart, Oblong, Oval, Round, Square**.

Данные разделены на обучающую и тестовую выборки. Задача формулируется как многоклассовая классификация.

---

## Метод решения

Для решения задачи используются сверточные нейронные сети, реализованные и обученные с нуля. В рамках работы рассмотрены и сравнены несколько архитектур:

- базовая CNN (SimpleCNN);
- улучшенная версия с нормализацией и регуляризацией (ImprovedCNN);
- архитектуры с механизмами внимания Squeeze-and-Excitation (AttentionCNN);
- углублённые модели с двойными сверточными блоками и расширенной регуляризацией.

В финальных версиях применяются:
- SE-блоки для каналного внимания;
- функция активации SiLU;
- Batch Normalization и Dropout;
- оптимизация с использованием AdamW и планировщика OneCycleLR;
- label smoothing для повышения устойчивости классификации.

---

## Оценка качества

Качество моделей оценивается с использованием следующих метрик:
- accuracy;
- precision, recall, F1-score;
- матрица ошибок (confusion matrix).

Проводится сравнительный анализ поведения моделей и структуры ошибок для разных архитектур.

---

## Результаты

Базовая модель демонстрирует точность классификации порядка **28%**.  
По мере усложнения архитектуры и внедрения механизмов внимания качество классификации увеличивается до **75–77%** на валидационной выборке. Финальные модели показывают более равномерное распределение ошибок и снижение путаницы между визуально близкими классами.

---

## Выводы

Результаты работы показывают, что последовательное архитектурное усложнение и применение механизмов внимания существенно повышают качество классификации формы лица. Использование SE-блоков, современных функций активации и продвинутых стратегий обучения позволяет добиться устойчивых результатов даже без применения предобученных моделей.
